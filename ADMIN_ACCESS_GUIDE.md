# EDUSANNA Admin Dashboard Access Guide

## Overview
Edusanna provides TWO secure methods to access the Admin Dashboard. Choose the method that works best for your security needs.

---

## METHOD 1: Direct Login (Easiest Method)

### Step-by-Step Instructions:

1. **Navigate to Login Page**
   - Go to: `https://your-edusanna.com/login`
   - Or click "Login" button on the homepage

2. **Enter Admin Credentials**
   - Email: `africanedusanna@gmail.com`
   - Password: `admin#1`

3. **Click Login Button**
   - You will be automatically redirected to `/admin`
   - The admin dashboard will load with full access

4. **Verify Admin Status**
   - You should see "Admin Dashboard" in the header
   - All admin panels will be visible and functional

---

## METHOD 2: Secret Admin Access (Advanced Security)

This method requires a special gesture and verification codes for additional security.

### Step-by-Step Instructions:

1. **Navigate to Homepage**
   - Go to: `https://your-edusanna.com/`

2. **Tap the Edusanna Logo (Secret Gesture)**
   - Look for the Edusanna logo in the top-left corner
   - Tap it **exactly 17 times**
   - A secret modal dialog will appear after 17 taps

3. **Enter First Verification Code**
   - When the modal appears, you'll see: "Enter Code 1"
   - Type: `Edusanna#1`
   - Click "Continue"

4. **Enter Second Verification Code**
   - You'll see: "Enter Code 2"
   - Type: `ES1`
   - Click "Verify"

5. **Access Granted**
   - Upon successful verification, you'll be redirected to the admin dashboard
   - All admin features will be available

### Security Notes for Secret Access:
- If you enter the wrong code, you'll see an error message
- You can try again immediately
- The 17-tap requirement prevents accidental access
- Two-code verification adds an additional layer of security

## TROUBLESHOOTING

### Issue: "Invalid Credentials" Error on Login
**Solution:** Double-check the email and password
- Email: `africanedusanna@gmail.com` (case-sensitive)
- Password: `admin#1` (exact match)

### Issue: Secret Modal Not Appearing
**Solution:** Make sure you're tapping exactly 17 times
- Count carefully: tap 1, 2, 3... up to 17
- Tap the logo in the navigation bar
- Modal should appear after tap 17

### Issue: Verification Code Error
**Solution:** Check code entry carefully
- Code 1: `Edusanna#1` (with capital E and #)
- Code 2: `ES1` (capital letters, number 1)
- Make sure there are no extra spaces

### Issue: Dashboard Not Loading
**Solution:** Try these steps:
1. Refresh the page (Ctrl+R or Cmd+R)
2. Clear browser cache
3. Try a different browser
4. Check internet connection
