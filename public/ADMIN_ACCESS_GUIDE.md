# Edusanna Admin Dashboard - Access Guide

## Admin Dashboard Access Methods

### Method 1: Secret Logo Tap Access (Recommended for Security)

1. Navigate to the Edusanna home page (/)
2. **Tap the Edusanna logo exactly 17 times** in the top-left navigation
3. A secret admin access modal will appear
4. Enter the first code when prompted: **`Edusanna#1`**
5. Enter the second code when prompted: **`ES1`**
6. You'll be automatically redirected to the admin dashboard

**Security Note:** This method ensures only users who know the secret codes and tap sequence can access the admin panel, protecting your platform from unauthorized access.

---

### Method 2: Direct Login Via Admin URL

1. Navigate directly to: `/admin`
2. If not already authenticated, you'll be redirected to the login page
3. Log in with your admin credentials:
   - **Email:** `africanedusanna@gmail.com`
   - **Password:** `admin#1`
4. You'll be granted access to the admin dashboard

**Note:** Ensure you have proper authentication configured before using this method.

## Admin Dashboard Features

### Overview Tab
- **System Status:** Real-time platform health monitoring
- **Quick Actions:** Fast access to settings, notifications, and analytics
- **Key Metrics:** Total users, active courses, completion rates, enrollments

### Completions Tab
- **Course Completion Tracking:** View all completed courses with full details
- **User Information:** 
  - Full name and email
  - WhatsApp contact number
  - Country and city location
- **Course Details:**
  - Course name and completion level (Certificate/Diploma)
  - Exam/assessment score
  - Start date and completion date
  - Progress timeline

### Users Tab
- **User Management:** View all registered users
- **Access Control:** Edit user access permissions
- **User Statistics:** Enrollment count, completion count, join date
- **User Status:** Active/inactive user management
- **Edit Access:** Modify user privileges and platform access

### Courses Tab
- **Course Catalog:** Browse all 140+ available courses
- **Search & Filter:**
  - Search courses by name
  - Filter by Certificate/Diploma level
- **Enrollment Tracking:** View enrollment numbers per course
- **Dual-Level View:** See which courses offer both Certificate and Diploma options

### Feedback Tab
- **User Ratings:** View all course ratings (1-5 stars)
- **Feedback Review:** Read detailed user feedback and reviews
- **Rating Management:** Edit or delete ratings as needed
- **Course Quality Monitoring:** Track course satisfaction through user reviews
- **Timestamp Tracking:** See when each review was submitted

## Support & Troubleshooting

### Access Issues
- **Can't access admin dashboard?** Verify your credentials or try the logo tap method
- **Forgotten password?** Use the password recovery option on the login page
- **Logo tap not working?** Ensure you're on the home page and tapping the logo in the header

### General Issues
- Check your internet connection
- Clear your browser cache and cookies
- Try accessing from a different browser
- Ensure Supabase connection is active
